# KIEMSOAT_RAVAO

Ứng dụng kiểm soát và phân tích dữ liệu y tế.

## Tính năng chính

- Phân tích dữ liệu với form N_TX_SQL
- Dự đoán và lọc dữ liệu theo mẫu
- Import/Export Excel
- Tích hợp DevExpress UI
- Kết nối SQL Server

## Công nghệ

- **Framework**: .NET Framework (Windows Forms)
- **UI**: DevExpress v24.2
- **Database**: SQL Server
- **Excel**: ClosedXML, iText

## Build

Mở `KIEMSOAT_RAVAO.sln` bằng Visual Studio 2019+ và nhấn `Ctrl+Shift+B`
